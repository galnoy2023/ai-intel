## 执行流程

### 1. 收集信息
```
1.1 访问美国来源
    - OpenAI News
    - Anthropic News
    - Google AI Blog
    
1.2 访问日本来源
    - METI
    
1.3 访问中国来源
    - 中国网信网
    - 北京智源
```

### 2. 验证来源
```
2.1 确认有原始链接
2.2 确认来源可信
2.3 排除不可靠信息
```

### 3. 判断级别
```
3.1 是否属于"必看"
3.2 是否属于"重要"
3.3 剩余归为"一般跟进"
```

### 4. 提炼内容
```
4.1 100字简介
4.2 为什么值得看
4.3 对我的启发
```

### 5. 输出
```
5.1 推送手机简报（必看+重要）
5.2 写入飞书多维表格（全部）
```

## 字段映射

飞书多维表格字段：
| 字段 | 说明 |
|------|------|
| 日期 | YYYY-MM-DD |
| 推送时段 | 早报/晚报 |
| 国家 | 美国/日本/中国 |
| 级别 | 必看/重要/一般跟进 |
| 主题分类 | 大模型发布/多模态/Agent等 |
| 标题 | 信息标题 |
| 100字简介 | 摘要 |
| 为什么值得看 | 判断理由 |
| 对我的启发 | 业务价值 |
| 来源机构 | 来源 |
| 原始链接 | URL |
| 来源类型 | 政府/官方博客等 |
| 是否建议立刻阅读 | 是/否 |
| 是否建议后续跟踪 | 是/否 |
| 可关联业务 | 游戏/运营/AI自动化等 |
| 状态 | 待读/已读/待跟进 |
| 备注 | 补充 |
